📌 Como publicar as alterações

  1. Clique em Carregar produtos.json existente para importar o arquivo atual (opcional se estiver começando).
  2. Adicione, edite ou remova produtos usando o formulário e a tabela.
  3. Clique em Exportar produtos.json para baixar o arquivo atualizado.
  4. Substitua o arquivo data/produtos.json no seu servidor pelo arquivo baixado.
  5. Pronto! A loja passa a exibir os novos dados no próximo acesso.
